= g_auto_completer

http://github.com/midas/g_auto_completer/tree/master


== DESCRIPTION:

A Guilded (http://github.com/midas/guilded/tree/master) Rails component that generates an auto complete
text box.


== FEATURES:


== INSTALL:

  sudo gem install midas-g_auto_completer


== USAGE:




== REQUIREMENTS:

* Rails >= 2.2.0
* Guilded >= 0.1.4 (http://github.com/midas/guilded/tree/master)
